# template-flowtype

[![npm](https://img.shields.io/npm/v/template-flowtype.svg)](https://www.npmjs.com/package/template-flowtype)

- Babel v7 (with `@babel/preset-env`)
- Flow
- ESLint (standardjs)
- MIT License

## Requirements

- [sao.js](https://github.com/saojs/sao)

## Usage

```console
$ sao npm:template-flowtype [directory]
```
